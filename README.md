# Scriportal - Modern University Registration Prototype

A modern, responsive, and user-friendly frontend prototype for the Tunisian University Registration system ("inscription.tn").

## Overview

**Scriportal** reimagines the student registration experience with a focus on usability, accessibility, and modern aesthetics. This prototype demonstrates a streamlined workflow for students to:
- **Login/Signup** with secure authentication flows.
- **Dashboard** to view personal and academic info.
- **University Email** management with integrated service discovery.
- **Payment** simulation for registration fees.
- **Multilingual Support** (French, Arabic, English).
- **Dark Mode** for better accessibility.
